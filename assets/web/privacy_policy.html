Collected Data

This bot collects and stores some data to function.

The following information is stored about the servers the bot is in:
  • The server_id
  • The server_name
  • The server_owner_id
  • The date the bot joined the server
  • The member_count of the server
  • The locale of the server

Settings from the server owner running /settings are stored. These settings are:

  • The role to ping when announcing starting streams
  • The channel to announce starting streams
  • Whether announcements for each platform are enabled

Command usage information is stored:

  • The server_id in which the command was run
  • The user_id of the member who ran the command
  • The date and time the command was run
  • The command which was run
  • The options that were used with the command

Abuse of the bot may result in a user or server being added to a blacklist. The blacklist contains:

  • The discord_id of the blacklisted user/server
  • The date_added to the blacklist
  • The expiry_date of the blacklist entry
  • The reason for being blacklisted

    • Backups and Data Restoration

    Backups are stored with Cloudflare and are encrypted before being uploaded.

    In the event of data loss and database restoration, the bot will check the database against connected servers and remove each server_id that removed the bot in the interim period between data loss and restoration.

    Logs

    Timestamped logs are stored locally. Logged events do not include any extra information than is outlined in Collected Data.

    Data Retention

    Logs are kept for 28 days.

    Server information and settings are kept for as long as the bot remains in the server.

    Blacklist data is kept indefinitely.

    Command usage data is kept for 3 months.

    Backups are kept for 28 days. Backups include the information outlined in Collected Data.

    Data Usage

    • Server and settings data are used to perform the announcement role of the bot.
    • Command data is used for aggregate statistics to improve services.
    • Blacklist data is used to prevent abuse of the bot.

    Data Sharing

    Backed up data is stored with Cloudflare. The data is encrypted before uploading and is stored in the European Union. No data is shared with any other third party.

    Data Removal

    To have your data be removed, remove the bot from your server.

    • Settings, server data and command data will be deleted within 24 hours.
    • Log data will be deleted after 28 days.
    • Backup data will be deleted after 28 days.
    • Blacklist data will be retained indefinitely.

    Contact

    If you have any concerns, you can join the Official Support Discord Server.
